<h2 align="center">
    How to install
</h2>

<h3>
  - Step 1
</h3>
<p>
	Download this repository with git	
</p>

```
$ git clone https://github.com/ogabriell/lousavirtual && cd lousavirtual
```
<h3>
  - Step 2
</h3>
<p>
	install the dependencies with npm
</p>

```
$ npm i express && npm i socket.io
```
<h2 align="center">
    How to start
</h2>
<p>
	To start the project just run this code
</p>

```
$ npm start
```
